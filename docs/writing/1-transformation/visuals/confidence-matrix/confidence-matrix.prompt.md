# Confidence-Graded Future Predictions

Three-tier matrix showing AI predictions by confidence level (2025-2035).

## Data Points

**Near Certainty (80%+):**
- Ticketing AI → 90%+
- Crowd monitoring standard
- Dynamic pricing → 70%+

**Probable (60-80%):**
- Personalized itineraries expected
- Predictive ops → 10-15%
- Regulatory frameworks stabilize

**Speculative (<60%):**
- Generative AI mainstream
- Small festivals → 75%+
- Full adaptive integration

## Style

- Colors: Gradient from deep purple (#6B46C1) near-certain to light coral (#FF6B6B) speculative
- White space: 40%+ composition
- Typography: Sans-serif, 16pt minimum
- Context: Embedded (textbook, no title)
- Accessibility: Confidence % + icon + color
- Festival theme: Subtle stage silhouettes

## Structure

Three horizontal tiers stacked vertically. Top tier (80%+) solid/bold, middle tier (60-80%) medium, bottom tier (<60%) light/transparent. Each tier shows 3 predictions with confidence percentage and year.
