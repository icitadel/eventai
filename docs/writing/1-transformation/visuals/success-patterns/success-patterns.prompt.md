## Visual Structure

### Four Quadrant Layout

Each quadrant represents one success pattern. Use clear dividers to separate sections while maintaining visual unity.

#### Quadrant 1: Bounded Use Cases Win (Top Left)

**Icon:** Target/bullseye with checkmark, or focused magnifying glass
**Color:** Deep Purple (#6B46C1)
**Title:** "Bounded Use Cases Win"
**Principle:** "Specificity enables success"

**Content:**
- ✓ **Success:** "Predict crowd congestion 10 min in advance"
  - Clear objective, measurable
- ✗ **Failure:** "Deploy AI to make festivals better"
  - Vague, unmeasurable

**Bottom line:** "Vague goals → vague results"

#### Quadrant 2: Transparency Builds Trust (Top Right)

**Icon:** Megaphone/announcement symbol, or open hand/handshake
**Color:** Sky Blue (#4299E1)
**Title:** "Transparency Builds Trust"
**Principle:** "Covert deployment backfires"

**Content:**
- ✓ **Success examples:**
  - Paris 2024: Public signage
  - Bonnaroo: Push notifications
  - DICE: User toggle options
- ✗ **Failure:** Taylor Swift 2018 (60K secretly scanned)

**Bottom line:** "40+ festivals banned facial rec after public pressure"

#### Quadrant 3: Humans in the Loop (Bottom Left)

**Icon:** Human figure + AI symbol connected, or person at dashboard
**Color:** Warm Sunlight (#F6AD55)
**Title:** "Humans in the Loop"
**Principle:** "AI recommends, humans decide"

**Content:**
- ✓ **Success examples:**
  - Qcue: 95% human review
  - NEC: Decision support (not autonomous)
  - EntertainmentLAB: Artist approval required

**Bottom line:** "AI augments expertise, doesn't replace it"

#### Quadrant 4: Phased Rollout Reduces Risk (Bottom Right)

**Icon:** Steps/stairs going upward, or three-stage progression
**Color:** Electric Coral (#FF6B6B)
**Title:** "Phased Rollout Reduces Risk"
**Principle:** "Test → Validate → Scale"

**Content:**
- **Year 1:** Monitor mode (collect data, build literacy)
- **Year 2:** Activate (A/B test, measure outcomes)
- **Year 3:** Scale (expand what works, kill what doesn't)

**Examples:** Tomorrowland, Bonnaroo, Paris 2024

**Bottom line:** "Mitigates risk, builds capability"

### Bottom Message Section

**Position:** Below four quadrants, centered
**Text:** "The technology works. The hard part is organizational readiness, clear use cases, and realistic expectations."
**Color:** Slate Gray (#2D3748)
**Size:** 14-16pt
**Style:** Italics, grounded, practical tone

## Visual Elements

⚠️ CRITICAL FESTIVAL CONTEXT:
- **Illustrated festival crowd visible in background** (subtle, 10-20% opacity)
- Festival stage, tents, or outdoor structures
- Wristbands on attendees
- Control room dashboard (for "Humans in Loop" quadrant)
- Festival operations context (staff with tablets, monitoring screens)

**Quadrant Icons:**
- One distinct icon per quadrant, matching quadrant color
- Simple, clean outline style
- Icons should feel festival-relevant (not generic business/tech)

**Success/Failure Indicators:**
- ✓ Green checkmarks for success examples
- ✗ Red X marks for failure examples (minimal, not dominant)
- Keep positive emphasis (success-oriented, not failure-focused)

## STANDARD Density Requirements

⚠️ CRITICAL WHITE SPACE:
- Target 30% white space composition
- Generous padding: 48-64px margins all sides
- 24-32px spacing between quadrants
- 16-24px padding inside each quadrant

❌ DO NOT ADD:
- Decorative borders around entire graphic or quadrants (subtle dividers OK)
- Gradient backgrounds (solid colors only)
- Multiple icons per quadrant (one icon max)
- Long explanatory paragraphs (keep to bullet points, 1-2 lines each)

✅ KEEP IT BALANCED:
- Four quadrants equally sized and visually weighted
- Clear dividers but unified composition
- Visual hierarchy: Title → Quadrants → Bottom message
- 30-60 second comprehension time

## Target Scan Time

30-60 seconds to understand:
1. Four distinct success patterns
2. Principle behind each pattern
3. Real-world success and failure examples
4. Bottom line: Implementation discipline matters

## Accessibility

- Text contrast minimum 4.5:1 (WCAG AA)
- Information not color-dependent (icons + text + color for each quadrant)
- Each quadrant distinguishable by icon AND title (not just color)
- Minimum 11pt body text, 14pt for titles

⚠️ REQUIRED - DO NOT SKIP:
- **Illustrated festival crowd visible in background**
- Festival stages, tents, outdoor venue structures
- Control room dashboard or monitoring equipment (for "Humans in Loop")
- Wristbands on attendees
