## Critical Findings

### üî¥ Issue #1: Title Generation Despite Prompt Constraints

**Problem:**
Variants #6 and #9 generated titles on infographic despite:
- Prompt removal of "Context: Embedded (textbook)" line
- Prompt removal of "Accessibility" and "Icons" specifications
- Intent to create clean, minimal infographics

**Impact:**
- Makes these variants unsuitable for embedded use
- 20% of new variants have this issue (2 out of 10)

**Root cause:**
Gemini's image generation may default to including titles unless EXPLICITLY constrained with negative instruction.

**Recommendation:**
Add to prompt Structure section:
```markdown
**CRITICAL: NO TITLE ON INFOGRAPHIC.**
- Title will be provided in figure caption
- Infographic should show data and labels only
- Do not add "Decision Framework: Before You Deploy" or any title text
```
