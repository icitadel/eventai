# Decision Framework: Before You Deploy

Pre-deployment assessment framework with three questions and four principles.

## Data Points

**Three Questions (Assess First):**
- Value clear to attendees?
- Can opt out freely?
- Preserves core values?

**Four Principles (If Yes):**
- VALUE FIRST
- SYMMETRIC CONSENT
- RESTRAINT (1-3/day max)
- ACCOUNTABLE

**Outcome Indicators:**
- Green: DEPLOY
- Yellow: REVISE
- Red: DON'T DEPLOY

**Key Threshold:**
- Control + Transparency + Restraint
- +40 NPS shift from intrusive to helpful

## Style

- Colors: Deep purple (#6B46C1), electric coral (#FF6B6B), bright green (#28A745), amber (#FFA500), red (#DC3545)
- White space: 40%+

## Structure

**NO decorative arrows or flow indicators.** Clean boxes and clear organization only.

**Top section - Two columns:**

Left column: "THREE QUESTIONS (ASSESS FIRST)" header in purple. Three questions in purple boxes, each with icon + text, stacked vertically.

Right column: "FOUR PRINCIPLES (IF YES)" header in coral. Four principles in coral boxes, each with icon + bold principle name, stacked vertically.

**Bottom section - Center aligned:**

"OUTCOME INDICATORS" - Three circular indicators arranged horizontally: green circle with checkmark ("DEPLOY"), yellow/amber circle with refresh icon ("REVISE"), red circle with X ("DON'T DEPLOY").

**Layout:** Clean box-based design. Questions and principles are separate columns at the top. Outcome indicators centered at bottom. No connecting arrows, no flow lines - just organized content in clearly labeled sections.
