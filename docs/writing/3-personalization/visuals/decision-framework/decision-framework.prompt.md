# Decision Framework: Before You Deploy

Pre-deployment assessment framework with three questions and four principles.

## Data Points

**Three Questions (Assess First):**
- Value clear to attendees?
- Can opt out freely?
- Preserves core values?

**Four Principles (If Yes):**
- Value First
- Symmetric Consent
- Restraint: 1-3/day max
- Accountable algorithms

**Outcome Indicators:**
- Green: Deploy
- Yellow: Revise
- Red: Don't deploy
