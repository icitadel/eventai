# Bonnaroo Engagement Gap: 86% Downloaded vs 20% Engaged

Visual showing the stark gap between app downloads and actual feature engagement.

## Data Points

**App Downloads:**
- 86% of attendees
- 60,000+ people
- Near-universal adoption

**Active Engagement:**
- 20% actually used it
- 12,000 active users
- 66-point gap

**The Gap:**
- 66 percentage points
- Availability ≠ acceptance
- Download ≠ engagement

## Style

- Colors: Deep purple (#6B46C1), electric coral (#FF6B6B), neutral gray (#6B7280)
- White space: 40%+
- Typography: Clean sans-serif, 18pt minimum, 32pt statistics
- Format: 1280×720 webp, landscape

## Structure

Side-by-side comparison bars. Left bar: 86% downloads (full height, purple). Right bar: 20% engagement (partial height, coral). Large "66-point gap" annotation between bars. Minimal text, maximum visual impact.
