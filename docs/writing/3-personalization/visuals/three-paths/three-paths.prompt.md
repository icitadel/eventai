# Three Paths to Organic Personalization

Three-column comparison of successful festival personalization approaches.

## Data Points

**Bonnaroo (Restraint):**
- Year 1: Zero sponsors
- 3.15 notifications/day
- 20% engagement rate
- Build trust first

**Meow Wolf (Collective AI):**
- Aggregate tracking only
- No individual IDs
- Environment responds
- Personalization without surveillance

**SXSW (Gamification):**
- Active opt-in only
- Leaderboards & badges
- Users control data
- Explicit value exchange
