# Helpful vs Intrusive Personalization

Two-column comparison showing what makes personalization helpful versus intrusive.

## Data Points

**Helpful Features:**
- Personalized schedules
- Interactive maps
- Emergency alerts
- +45 NPS impact
- User control

**Intrusive Features:**
- Location tracking
- 12+ notifications/day
- Biometric scanning
- -40 NPS impact
- Surveillance feel

**The Threshold:**
- Control + transparency
- +40 NPS shift
- Adoption vs rejection
