## How to Generate Each Visual (Quick Workflow)

### For Each Visual (9 Steps, ~15-30 min)

**Step 1: Create NotebookLM Notebook**
- Go to https://notebooklm.google.com
- Click "+ New notebook"
- Name it: `EventAI VIS-5.X - [Visual Title]`

**Step 2: Upload Source Document**
- Click "+ Sources"
- Upload: `[visual-name].content.md`
- Wait for processing

**Step 3: Apply Style Prompt**
- Copy the full prompt from `.instructions.md` Step 3
- Paste in NotebookLM chat
- Wait for acknowledgment

**Step 4: Generate Visual**
- Type: `Generate this as an infographic`
- Select: High detail, Landscape, Largest size

**Step 5: Review Quality**
- Check against checklists in `.instructions.md` Step 5
- Verify statistics match source
- Verify colors/style match requirements

**Step 6: Iterate if Needed**
- If minor issues: Use adjustment prompts from `.instructions.md` Step 6
- If major issues: Revise prompt and regenerate

**Step 7: Download & Save**
- Download PNG from NotebookLM
- Rename to: `analytics-infographic-[visual-name].png`
- Save to: `/[visual-name]/` folder

**Step 8: Optimize (Optional)**
- Convert PNG to WebP for web use
- Command: `convert [file].png -quality 85 [file].webp`

**Step 9: Document Results**
- Create: `[visual-name]-ANALYSIS.md`
- Note: What worked, what needed adjustment, lessons learned
