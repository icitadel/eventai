## Key Learnings & Prompt Evolution

### Critical Prompt Changes That Led to Success:

**1. White Background Enforcement (Round 1 → 2):**
```markdown
# Added:
"PURE WHITE #FFFFFF BACKGROUND (NOT gray, NOT cream, NOT #F8F8F8)"
```
**Result:** Achieved pure white background, improved from 89% → 94%

**2. EU AI Act Visual Style (Round 2 → 3):**
```markdown
# Added:
Quadrant visual treatment (inspired by EU AI Act pyramid style):
- Saturated colored borders (thick, 4-6px)
- Light pastel fills (#E6F7ED, #FFE6E6, #FFF4E6, #FFEDE6)
- Black text on pastel backgrounds (vs. white on saturated)
```
**Result:** Dramatically improved readability, user approved visual style

**3. Remove Q-Labels (Round 3 → 4):**
```markdown
# Added to AVOID:
"quadrant numbers (Q1/Q2/Q3/Q4)"

# Removed from DATA POINTS:
Q1/Q2/Q3/Q4 references

# Updated color-blind accessibility:
"NO quadrant numbers (Q1/Q2/Q3/Q4) - not meaningful data"
```
**Result:** Fixed rendering artifacts, cleaner design, improved from 94% → 96%

**4. Color-Blind Accessibility:**
```markdown
# Triple encoding:
- COLOR + ICONS + TEXT LABELS (not Q-numbers)
- Distinct icons: ✅ ❌ ⚠️
- Text labels: APPROPRIATE, ILLEGAL, CONSTRAINED, CONDITIONAL
```
**Result:** Fully accessible without color dependence
