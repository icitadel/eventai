# NotebookLM Capabilities (2025)

Comprehensive documentation of NotebookLM's features, focusing on downloadable rich content generation for the Lemmy system.

## Overview

NotebookLM is Google's AI-powered research and content generation platform that transforms source materials into multiple rich media formats. As of 2025, it's evolved into a comprehensive content creation engine with audio, video, slide deck, and infographic generation capabilities.

**Primary Value for Lemmy**: Transforms research into professional multi-format content with minimal manual effort.

---

## 1. Audio Overview

Transform sources into AI-generated podcast-style conversations between two hosts.

### Formats (September 2025+)
- **Deep Dive**: Comprehensive exploration with back-and-forth discussion
- **Brief**: Quick 5-minute summary of key points
- **Critique**: Critical analysis and evaluation
- **Debate**: Opposing viewpoints and argumentation
- **Lecture** (Experimental): Single continuous explanation structured like a class session

### Length Options
- **Shorter**: ~5 minutes
- **Default**: ~10 minutes
- **Longer**: ~20 minutes
- **Lecture Long**: ~30 minutes (Lecture mode only)

### Customization Options
- **Topic Focus**: Guide what hosts emphasize
- **Expertise Level**: Adjust technical depth
- **Voice Tone**: Control presentation style (March 2025+)
- **Section Prioritization**: Focus on specific sources
- **Language**: 80+ languages supported (August 2025+)

### Interactive Features
- **Join Conversation**: Ask follow-up questions via voice (English only currently)
- **Request Details**: Dive deeper on specific topics during playback

### Download Format
- Audio file (MP3/downloadable format)
- Available immediately after generation

## 2. Video Overview

AI-narrated slide videos with contextual illustrations powered by Nano Banana (Gemini's image generation model).

### Formats
- **Explainer**: Structured, comprehensive video based on sources
- **Brief**: Bite-sized video for quick core ideas

### Visual Styles (18+ only)
- Classic
- Whiteboard
- Watercolor
- Retro Print
- Heritage
- Paper-craft
- Kawaii
- Anime

**Note**: Under-18 users have access to Video Overviews with age-appropriate visual styles.

### Features
- **AI-Generated Illustrations**: Nano Banana creates contextual, beautiful illustrations from your sources
- **Pulls Visual Content**: Automatically extracts images, diagrams, quotes, and numbers from documents
- **Steering Prompts**: Customize focus on specific sources or topics
- **Multi-language**: Generate in 80+ languages

### Download Options
- **MP4 file**: 1080p resolution, under 10 MB
- **Embed Code**: iframe code for Google Sites or WordPress

### Launch Date
Rolled out to all users starting week of August 4, 2025

## 3. Slide Deck

Professional presentation slides generated from your sources.

### Export Options
- **PDF Download**: Static, non-editable format (primary export)
- **Individual Images**: Right-click save on any slide
- **Shareability**: Generate shareable links with access control
- **Full-screen Slideshow**: Present directly in NotebookLM

### Customization
- **Steering Prompts**: Guide content focus
- **Source Selection**: Choose which sources to emphasize

### Editing Limitations
‚ö†Ô∏è **Important**: Slide decks are **NOT directly editable** in NotebookLM. You can only modify by:
1. Changing your source materials
2. Adjusting prompts and regenerating
3. Exporting PDF and editing in PowerPoint/Google Slides

### Workflow
1. Generate slide deck in NotebookLM
2. Download as PDF
3. Import PDF to PowerPoint or Google Slides for editing
4. Make final adjustments in external tool

## 5. Deep Research

Automated comprehensive research assistant that browses the web on your behalf.

### Key Capabilities
- **Research Plan Creation**: Automatically structures research approach
- **Web Browsing**: Autonomous website exploration
- **Source Recommendations**: Suggests relevant articles, papers, websites
- **Report Generation**: Synthesizes findings into detailed, source-grounded reports
- **Background Operation**: Runs while you continue working

### Research Modes
- **Deep Research**: Full briefing with in-depth analysis and high-quality sources (runs in background)
- **Fast Research**: Quick search for immediate source review and import

### Search Options
- **Web Search**: Entire internet (default)
- **Google Drive**: Search only within your Drive files

### Workflow
1. Go to "Sources" panel
2. Choose "Web" as source type
3. Select "Deep Research" from dropdown
4. Enter research question
5. Choose Deep or Fast mode
6. Review generated report
7. Add report AND sources to notebook

### Launch Date
Added November 2025

## Platform Strengths Summary

### What NotebookLM Does Best
1. **Multi-format Content Generation**: One research source ‚Üí multiple output formats
2. **Rich Media Creation**: Audio, video, visuals without specialized tools
3. **Source-Grounded Content**: Always tied to your materials, not generic AI
4. **Background Processing**: Generate while continuing other work
5. **Downloadable Outputs**: All content can be exported and integrated elsewhere

### What NotebookLM Doesn't Do Well
1. **Direct Editing**: Can't edit generated slides/infographics directly
2. **Fine-grained Visual Control**: Limited customization of specific visual elements
3. **API Access**: No official API (requires Playwright automation)
4. **Batch Operations**: Must generate one item at a time (unless scripted)
