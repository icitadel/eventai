## Workflow Integration

### With Claude Code

**Typical Flow**:
1. **Claude Code**: Prepares content, identifies template needs
2. **Human**: Selects appropriate template from this index
3. **Claude Code**: Fills template placeholders with specific content
4. **Human**: Copies filled template to target platform (NotebookLM, Gemini, etc.)
5. **Platform**: Generates content
6. **Human**: Downloads/exports generated content
7. **Claude Code**: Integrates into final documentation

### Future Automation

**Playwright Integration** (planned):
- Claude Code generates filled prompts
- Playwright script uploads to NotebookLM
- Automated generation triggering
- Automatic download of outputs
- Claude Code receives files for integration

**Gemini API Integration** (planned):
- Claude Code generates Nano Banana prompts
- Direct API calls for image generation
- Automated download and integration
- No manual platform interaction needed
