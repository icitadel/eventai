# Lemmy System Overview

A three-phase knowledge-to-output creation system showing the flow from corpus building to narrative extraction to output creation.

## Data Points

**Three Phases:**
- Lemmy → Corpus Building
- Taley → Narrative Extraction
- Tooly → Output Creation

**Three Knowledge Types:**
- Topic Knowledge
- Voice Knowledge
- Content Knowledge

**Key Benefits:**
- Reusability
- Quality Standards
- Collaboration
- Flexibility
- Reliability

## Style

**Aesthetic**: Schoolhouse Rock!-inspired educational design with owl and learning motifs

**Color Palette** (2-3 primary colors maximum):
- Wise Owl Purple (#6B46C1) - Lemmy phase marker
- Electric Coral (#FF6B6B) - Taley phase marker
- Soft Teal (#4ECDC4) - Tooly phase marker
- Cream background (#F7F7F2) - warm, not stark white
- Charcoal text (#3A3A3A) - not pure black

**Visual Style**:
- Hand-drawn quality with bold outlines (3-5pt stroke)
- Flat colors, minimal gradients
- Comic book art aesthetic
- Playful, approachable, educational optimism
- Simple geometric shapes for clarity

**Typography** (Concise tier - bold labels only):
- Headings: 24-32pt bold, playful sans-serif (Cooper Black style)
- Labels: 18-22pt, ALL CAPS for phase names
- Maximum 2 font families
- High contrast for accessibility (WCAG AA)

**Owl & Learning Motifs** (minimal usage):
- 1-2 simple owl silhouettes or icons
- Owl positioned as section marker or corner badge
- Simple arrow flows (hand-drawn style, slight wobble)
- Maximum 1 educational icon (book, light bulb, or connected nodes)

**Composition**:
- White space: 40%+ (generous breathing room)
- Central focal point with radial or horizontal flow
- Bold visual hierarchy
- Maximum 3 visual sections
- High contrast between elements

**Format**:
- 1280×720 webp, landscape (16:9 aspect ratio)
- No title on image
- **CRITICAL:** ALWAYS landscape orientation

**Accessibility**:
- Color contrast minimum 4.5:1 (WCAG AA)
- Icons paired with text labels
- Never rely on color alone for meaning

Left-to-right flow diagram showing three phases with arrows. Each phase labeled with name and purpose arrow. Three knowledge types shown as supporting layers below the phases. Key benefits listed as compact badges.
